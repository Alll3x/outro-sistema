# Para devs
  ## Comandos
   ### Para executar em modo de desenvolvedor 
    npm run dev
   ### Para auto commit 
    npm run commit 
   ### Para auto commit e autoplay
    npm run commitplay
   ### Para atualizar 
    npm run atualizar
