# Sistema básico com as funcionalidades
  - Cadastrar usuário com endereço
  - Cadastrar veículo do usuário
  - Criar uma ficha de manutenção 

# Para devs
  ## Para executar em modo de desenvolvedor 
    - npm run dev
  ## Para auto commit 
    - npm run commit
